# A4封筒作成ツール

A4サイズの紙から手作りで封筒を作成するためのツールです。
Web上で宛名と差出人を入力して、そのままA4サイズで印刷できます。

📍 **[アプリケーション](https://onfi.github.io/envelope/)**

## 使い方

### 1. 宛名を入力する
上部の「宛先」テキストエリアに宛名を入力します。
- 住所、名前など自由にレイアウト調整できます
- 改行を入れることで位置調整が可能です
- リアルタイムでプレビューに反映されます

**入力例：**
```
〒100-0001


東京都千代田区千代田1-1

日本 太郎 様
```

### 2. 差出人を入力する
下部の「差出人」テキストエリアに送付元の情報を入力します。
- 郵便番号、住所、氏名など必要な情報を入力してください
- こちらも改行で位置調整できます

**入力例：**
```
〒150-0002

東京都渋谷区2-2
日本 花子
```

### 3. 印刷する
「印刷する」ボタンをクリックして、ブラウザの印刷機能を開きます。
- 用紙サイズ：**A4**
- 向き：**横向き（ランドスケープ）**
- 余白：なし推奨

印刷してください。

### 4. 線に沿って折る
印刷した用紙を以下の順序で折ります。
1. **下側** の線に沿って折る
2. **右側** の線に沿って折る
3. **左側** の線に沿って折る

### 5. 中身を入れる
折った封筒の開き口に、手紙や書類などを入れます。

### 6. 封をする
右側に記載されている「のりしろ」部分にのりをつけて、封筒を閉じたら完成です。

## 機能

- 📝 リアルタイムプレビュー：入力すると即座にプレビューに反映
- 💾 自動保存：入力データをブラウザのローカルストレージに自動保存
- 🔄 データ復元：ページを再度開いた時に前回のデータを復元
- 🖨️ 印刷対応：そのままA4サイズで印刷可能

## 必要なもの

- ブラウザ（Chrome、Firefox、Safari など）
- A4用紙
- プリンター
- のり
